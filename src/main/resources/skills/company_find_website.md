# 公司官网查找

当用户明确要求「查找某公司官网」「找某某的官方网站」时，由本 Skill 处理。仅执行官网查找（含 Tool），返回公司名与官网 URL 等信息。若用户还要求「对网站做分析」「简要分析」，应由规划层拆成两个任务，本 Skill 处理查找，公司网站分析 Skill 处理分析。
